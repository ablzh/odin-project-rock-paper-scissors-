# The Odin Project: Rock Paper Scissors 

[This project](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors) is part of **The Odin Project** curriculum, created for practicing **JavaScript**. 

The goal is to build the game [Rock Paper Scissors](https://www.wikihow.com/Play-Rock,-Paper,-Scissors).

**How to play:**
1. Open the [web page](https://ablzh.github.io/odin-project-rock-paper-scissors/)
2. Open the web console (in Firefox: ctrl + shift + k)
3. Enter the figure via the prompt, the window with the input field in the browser
4. Monitor the round results in the console.
